# FMG

## Tools used

- Node: 16.15.1
- NPM: 8.12.1
- Docker: 20.10.16

## Running the app

### With Docker

- `./dc.sh`
- UI available at `localhost:3000`, API at `localhost:4000/graphql`

### NPM scripts

- `cd ./api && npm install && npm start`
- `cd ./web && npm install && npm start`
- UI available at `localhost:3000`, API at `localhost:4000/graphql`
